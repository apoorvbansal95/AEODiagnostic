# AEODiagnostic

Answer Engine Optimization -> Rank the products on basis of feedback given by different LLM's 

## What this project does

Runs a query (e.g. "best whey protein for fat loss") through multiple LLMs simultaneously,
collects their brand recommendations, scores each brand's AI visibility, and returns a
structured AEO (AI Engine Optimization) diagnostic report via a FastAPI backend.
